# Netskope BEAM

Behavioral Evaluation of Application Metrics (BEAM) is a Python library for detecting supply chain compromises by analyzing network traffic.

## Installation

Sync the BEAM repo, and then run:

```bash
pip install -e .
```

## Usage
Navigate to `beam/src` and run:

```bash
python beam
```
